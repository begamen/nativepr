Demo Installation

Extract package.
 
Copy the entire contents of the sepia folder to the location on your web server where you want your site to be accessed. If you want your primary domain to point to your install of Sepia Theme, copy everything to the root folder. If you want your site to be accessed as a subsection of your domain (for example: yourdomain.com/drupal) then simply rename the sepia folder to the desired name and upload to the root of your server.
 
Access your MySQL database and create a new DB (e.g. sepia_theme - Collation - utf8mb4_unicode_ci ).
 
Import the SQL file sepia.sql.zip to database for theme (e.g. sepia_theme).
The PhpMyAdmin might be helpfull for managing your databases.
 
Open the file settings.php  (sites/default/settings.php) and edit (at the bottom of  opened file,  add - your database name, user name and password ):
 
 
 
$databases['default']['default'] = array (
  'database' => 'sepia',
  'username' => 'root',
  'password' => 'pass',
  'prefix' => '',
  'host' => 'localhost',
  'port' => '3306',
  'namespace' => 'Drupal\\Core\\Database\\Driver\\mysql',
  'driver' => 'mysql',
);
 
 
Go to your-website/user and Login to your site with the username Admin and the password demo and visit admin/people to change the admin username and password again, visit admin/config/system/site-information to change the site-information again.

Go through the Drupal configuration found in admin/config options and update all the necessary information to reflect your website. Also be sure to visit /admin/config/media/file-system and update the Files Temporary directory .

Visit admin/config/development/performance.
 
Clear all cache.

Install theme on existing sites

First, download and enable the Twig tweak module.

The theme.zip file is the theme only package. You can use this theme to install in your existing Drupal site.

Just extract the theme.zip file, and upload to /sites/all/themes folder.

Please go to Admin > Appearance and enable the theme.

After that, you will be able to use predefined styles of the theme to build the Drupal site of your own.

Theme Seting

After you have installed the theme, you can change its settings  Appearance - Theme settings.

Here you can switch some elements of the website on/off, Show Preloader, Show scroll-to-top button, Dark version  as well as change the logo and favicon. 

Using grid system

The theme is based on Bootstrap framework

So what you need to know about using this grid system?

First of all, every column, or page section must have it's container. That is div with class "container". Inside of it there must be div or section with class "row". Inside one row, you put your columns. One row is actually one row visually. That means that column with class .col-md-12 will go inside one row. In row below that column can be another row that is containing for example .col-md-4 and col-md-8.


How do you combine columns?

As there are 12 columns, sum of it's classes numbers must be 12. That is, in one row can go col-md-7 and col-md-5, or col-md-3 and col-md-9, and so on. Important thing to remember is that inside one row can go columns which sum is 12. Every row has it's bottom margin, which is 80px. So it is important to remember that one row is including only one visual row of columns.
